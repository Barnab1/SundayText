 ## How can you get started ?

To use the project, follow the steps:

1- Clone the project 

    `git clone https://github.com/Barnab1/SundayText`

    `cd SundayText` && `cd churchText`

2- Open terminal and install dependencies

    `npm install`

3- Launch development server

    `npm start `

4- Open the App in browser

    `http://localhost:3000`
